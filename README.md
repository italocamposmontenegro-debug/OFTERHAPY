# Protocolo Orofacial (PWA Local)

Aplicación Web Estática Para GitHub Pages.

## Características
- 100% Local: Guarda En `localStorage`.
- Modo Día/Noche (Fondo Negro Por Defecto).
- Paleta: Púrpura + Dorado + Verde Pistacho.
- Exportación: JSON Individual, JSON Total, CSV Individual.
- Reporte Imprimible: “Imprimir / PDF” (Usa El Diálogo De Impresión Del Navegador).
- PWA: Instalación Opcional Y Caché Offline (Service Worker).

## Estructura
- `index.html`
- `styles.css`
- `app.js`
- `manifest.json`
- `service-worker.js`
- `assets/icon-192.png`
- `assets/icon-512.png`

## Despliegue En GitHub Pages
1. Suba Los Archivos Al Repositorio (Raíz).
2. En GitHub: **Settings → Pages**
3. Source: `Deploy from a branch`
4. Branch: `main` / Folder: `/ (root)`
5. Guardar Y Abrir La URL Que Entrega GitHub.

## Notas De Contenido
Algunos Ítems Están Con “Instrucción Pendiente” Para Evitar Inventar Pasos Clínicos No Especificados.
Puede Editar `ITEMS` En `app.js` Y Actualizar Instrucciones.
